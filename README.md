1. Class Mahasiswa
   Class Mahasiswa digunakan sebagai kelas data/objek yang menyimpan informasi tentang satu mahasiswa.
   Isi di dalamnya berupa atribut, constructor, setter/getter, dan method tampilkanData().
   Atribut yang digunakan:
   Atribut adalah variabel global yang dimiliki setiap objek Mahasiswa.
   Atribut	Tipe Data	Fungsi:
   1. nama	String	Menyimpan nama mahasiswa
   2. nim	String	Menyimpan nomor induk mahasiswa
   3. prodi	String	Menyimpan program studi mahasiswa

2. Class MainApp
   Class ManApp adalah kelas utama yang menjalankan program.
   Fungsi utama class ini:
   1. Menyediakan menu (tambah data, tampilkan data, keluar).
   2. Menggunakan ArrayList<Mahasiswa> untuk menyimpan banyak objek mahasiswa.
   3. Memakai Scanner untuk menerima input dari user.
   4. Membuat objek Mahasiswa menggunakan constructor lalu memasukkannya ke ArrayList.
   5. Memanggil method tampilkanData() untuk menampilkan informasi.
