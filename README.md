1. Class Mahasiswa
   Class Mahasiswa digunakan sebagai kelas data/objek yang menyimpan informasi tentang satu mahasiswa.
   Isi di dalamnya berupa atribut, constructor, setter/getter, dan method tampilkanData().
   Atribut yang digunakan:
   Atribut adalah variabel global yang dimiliki setiap objek Mahasiswa.
   Atribut	Tipe Data	Fungsi:
   - nama	String	Menyimpan nama mahasiswa
   - nim	String	Menyimpan nomor induk mahasiswa
   - prodi	String	Menyimpan program studi mahasiswa

2. Class MainApp
   Class ManApp adalah kelas utama yang menjalankan program.
   Fungsi utama class ini:
   - Menyediakan menu (tambah data, tampilkan data, keluar).
   - Menggunakan ArrayList<Mahasiswa> untuk menyimpan banyak objek mahasiswa.
   - Memakai Scanner untuk menerima input dari user.
   - Membuat objek Mahasiswa menggunakan constructor lalu memasukkannya ke ArrayList.
   - Memanggil method tampilkanData() untuk menampilkan informasi.
